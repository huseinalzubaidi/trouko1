لا احد يختار اسمه قبل الولادة فجميعنا نولد وتفرض علينا أسمائنا من قبل أحد الوالدين أو من غيرهم، وربما لأسمك دور في التأثير على حياتك، وللاسامي معاني تؤثر على صاحبه . هذه الدراسات على الأسماء ليست مصدراً موثوقا بشكل تام وليس من الضروري أن تكون صحيحة في جميع الحالات، حيث اعتمدت الدراسات على بيانات تنقيبية متعلقة بالأفراد وأسمائهم، ثم تم ترتيبها وفحصها وتدقيقها، وأجمعت الدراسات على أن نسبة الدقة أعلى من 60 بالمئة.

الأسماء باللغة العربية

أعرف شخصيتك بأول حرف من أسمك بحسب الدراسة العربية للحروف العربية :

الحروف ( ب، ث، ص، ض، ن، و، ي ) : يكون أصحابها متواضعون، و يمتازون بطيبة القلب، صبورون، يتسمون بالوفاء في الحبّ والعلاقات الاجتماعية والرزانة.


الحروف ( ت، ج، س، ز، ظ، ق، ك ) : يكون أصحابها محبون للخيال والتأمل، وعطوفون وسريعون التأثر، وهم بحاجه الى الحب والرومانسية.


الحروف ( أ، ذ، ش، ط، ف، م، ھ ) : يكون أصحابها عصبيون، ويتّسمون بالحيوية والنشاط، ومعجبون بنفسهم ونرجسيون ومتكبرين، وأذكياء وعاطفيون، ومثاليون في كثير من نواحي الحياة.


الحروف ( ح، خ، د، ر، ع، غ، ر، ل ) : يكون أصحابها كرماء و يتميزون بالبساطة، والكرم وحسن التصرّف، و الخيال الواسع.

الأسماء باللغة الانجليزية

أعرف شخصيتك بأول حرف من أسمك بحسب الدراسة الانجليزية للحروف الانجليزية :

الحرف ( A ) : يتميز بالمثابرة و هو شخص عملي. ينال ما يخطط له، مع هذا هو لا يتفاعل مع الأشخاص اللطيفين بالتعامل معه و غير صبور.


الحرف ( B ) : يتميز بالصبر وينتظر بصبر حتى يحقق ما يريده، يتحكم بمشاعره وأحاسيسه، يحب تلقي الهدايا , و يملك القدرة على خوض المغامرات.


الحرف ( C ) : يتميز بالأندماج مع البيئة المحيطة به و يمتلك شخصية اجتماعية قوية، حسّاس لكنه خبير في التحكّم بالعلاقات، و يمتاز بالصبر.


الحرف ( D ) : يتميز بالاخلاص وهو حساس، متحرر في التصرفات لكن بوعي وليس بغير عقلانية، مندفع المشاعر ويفقد أعصابه بسرعة.


الحرف ( E ) : يتميز بالأندماج مع البيئة المحيطة به و يمتلك شخصية اجتماعية قوية، يحبّ الصداقة، ويحبّ التحدّث والتعبير عن نفسه كثيراً.


الحرف ( F ) : يتميز بقوة الشخصية و الاستعراض فهو شخص مثاليّ وشجاع وأنيق.


الحرف ( G ) : يتميز بالنشاط و طول الباّل ، ويبحث عن الكمال دون كلل أو ملل.


الحرف ( H ) : يتميز بالكرم، وهو إنسان محبوب ورقيق وقوي، و حذر جداً في علاقاته.


الحرف ( I ) : يتميز بالمثابرة و البحث و المغامرة ، لكنه شخص صغير مخلص وعلاقاته الرسمية لا تطول.


الحرف ( J ) : يتميز بالذكاء و الناجح، مثاليّ في إنشاء العلاقات بسهولة.


الحرف ( K ) : يتميز بحفض الاسرار، وخجول، وانطوائيّ، وحسّاس، وعطوف وغالباً ما يكون جذاباً.


الحرف ( L ) : يتميز بالرومانسية، منجذب دائماً للحبّ، كما أنه يتمتّع بذكاء كبير.


الحرف ( M ) : يتميز بقوة الشخصية و الذكاء و لديه صعوبة في التعبير عن نفسه ، يهدف للفوز دائماً مهما كلفه الأمر.


الحرف ( N ) : يتميز بالعطف والأحساس، و يؤمن بالحرية.


الحرف ( O ) : يتميز بالأندماج مع البيئة المحيطة به و يحب المشاركة في النشاطات الاجتماعية، لكنه كتوم في إعلان نفسه ورغباته.


الحرف ( P ) : يتميز بالرومانسية فهو شخص شاعري ومدرك، و يتصف بالذكاء.


الحرف ( Q ) : يتميز بالنشاط فهو شخص يتمتّع بطاقة حركية كبيرة، كما يمتاز بالحماس والحبّ.


الحرف ( R ) : يتميز بالسلوك اللبق، وقادر على توجيه سلوكه وتصحيحه، يبحث عن شخص متوافق مع قدراته العقلية والنفسيّة.


الحرف ( S ) : يتميز بالروح المرحة في العمل و مع أصدقائه و افراد عائلته، و يغلب عليه طابع التملك والغيره، هو حسّاس وعطوف وكتوم، ويبرز عليه الانفعال بسرعة.


الحرف ( T ) : يتميز بالرومانسية و يحب الموسيقى والهدوء والضوء الخافت، ولا يهتمّ بكلام ونصائح الآخرين.


الحرف ( U ) : يتميز بالنشاط و الحماس، و يحتاج دائماً لمغامرات جديدة وإحساس بالحماس والحرية المطلقة.


الحرف ( V ) : يتميز بالحماس و التسرع ولا يطيق الانتظار، لا يخاف من الخطر.


الحرف ( W ) : يتميز بالثقة بالنفس و فخور بنفسه، يدعم نفسه بنفسه، حكيم ورزين في أمور حياته.


الحرف ( X ) : يتميز بقوة شخصيته ، ولديه القدرة على عمل علاقات في أكثر من وقت واحد.


الحرف ( Y ) : يتميز بالعطف والأحساس، ومستقلّ بشخصيته، يصر على اثبات شخصيته ونفسه بأنّه الأفضل.


الحرف ( Z ) : يتميز بالرومانسية و لا يظهر مشاعره بسهولة، واجتماعيّ الى حد كبير.
